# Kenito the Weather Man

Kenito the Weather Man is a robust, production-ready Telegram Bot that delivers automated, highly personalized weather updates and AI-generated practical advice to subscribers in the Philippines.

Built with performance and reliability in mind, this project integrates real-time weather data with large language models (LLMs) to provide context-aware recommendations, not just raw meteorological numbers.

## Key Features

* Automated Daily Broadcasts: Subscribers receive updates three times a day (6:00 AM, 12:00 PM, 6:00 PM PHT).
* AI-Powered Recommendations: The morning broadcast includes a professional advice generated by Groq's Llama 3.1 model based on current time and conditions.
* On-Demand Updates: Users can request real-time weather checks anytime via the /weather command.
* Persistent Subscriber Management: A Supabase PostgreSQL database handles user subscriptions, coordinates, and active status.
* Process Monitoring & Uptime: The server runs as a background daemon using PM2 for automated restarts on failures.

## Technical Stack

* Backend Runtime: Node.js (Deployed via Termux on Android)
* AI Model: Groq Llama 3.1-8b-instant
* Database: Supabase (PostgreSQL)
* Weather Data API: OpenWeatherMap API
* Telegram Bot API: node-telegram-bot-api
* Scheduling: node-cron
* Process Manager: PM2

## Usage

You can interact with the live bot on Telegram:
t.me/panahongamitsikenitobot

Or send the following commands:
* /start - Subscribe and setup your location.
* /weather - Get an on-demand, real-time weather update.
* /help - Show the interactive command menu.
* /stop - Unsubscribe from daily updates.

## Local Setup

To run this project locally, clone the repository, install dependencies, and create a .env file with your API keys:

1. git clone https://github.com/gofredo69/Kenito-the-Weather-Man.git
2. cd Kenito-the-Weather-Man
3. npm install
4. Create a .env file based on .env.example (add your actual keys).
5. npm start
